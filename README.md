# Abastecimiento - Milicic (GitHub Pages)

## Cómo publicar
1. Subí TODOS los archivos de este ZIP a la raíz del repo.
2. En GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
3. La home es `index.html` (Cumplimiento).

## Páginas
- index.html (Cumplimiento)
- analisis-mm.html
- demoras.html

## Importante
- GitHub Pages es sensible a mayúsculas/minúsculas (case-sensitive).
- Los CSV deben quedar con los mismos nombres que usa cada JS.
